```js
} from "@rent_avail/autocomplete"
```

```jsx live
<AutocompleteProvider apiKey={placesKey}>
  <Autocomplete label="Street Address" />
</AutocompleteProvider>
```

### Description

#### Style Props

`space, layout, border, color, flexbox, grid`

#### Component Props

_The Autocomplete component composes the input component, and takes all props that you can pass into that component._

<br />

| Name     | Type       | Description                                                                                 |
| -------- | ---------- | ------------------------------------------------------------------------------------------- |
| onSelect | `function` | Callback function firing when the user selects an address from the options provided.        |
| onClear  | `function` | Callback function firing when the user clears the selection by focusing on the input again. |
