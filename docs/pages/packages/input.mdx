### Description

Text inputs let users enter and edit text information. They are made up of a label, value, helper text, and optionally a leading / trailing icon.

#### Style Props

`space, layout, flexbox, grid, color, border`

### Props

| name     | type      | description                                                                      |
| -------- | --------- | -------------------------------------------------------------------------------- |
| label    | `string`  | Used to describe what the user should enter.                                     |
| icon     | `element` | Element positioned to the left of the input to give context to the value.        |
| required | `boolean` | Indicates that the input will error when the form is submitted if not validated. |
| error    | `string`  | Value for the error to display associated with the input.                        |
| disabled | `boolean` | Indicates if the input can be focused on by the user.                            |

### Usage

```jsx live
<Input label="Full Name" />
```
