### Description

Menus display a list of actions or choices on a temporary card. They offer a condensed way to present multiple actions and actions that a user won't always need to see.

#### Style Props

`space`

#### Menu Props

| name | type     | description                       |
| ---- | -------- | --------------------------------- |
| id   | `string` | Identifier used for accessability |
