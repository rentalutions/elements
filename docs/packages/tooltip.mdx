### Description

Tooltips offer greater context to an action or element on the page. Tooltips should never contain information needed to complete a task.

## Tooltip

This element should always wrap the tooltip component. It provides the show/hide state to the content as well as adding accesability.

#### Props

| Name | Type     | Description                                                                |
| ---- | -------- | -------------------------------------------------------------------------- |
| id   | `string` | A unique identifier for the tooltip, should not be duplicated on the page. |

## Tooltip Target

This element wraps whatever the user will interact with to toggle the tooltip showing. It's child can be any valid element.

## Tooltip Content

This element contains the extra information that the user will have to toggle on to view. It's accesable to screen readers and can through the focus attribute.
