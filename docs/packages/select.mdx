```js
import { Select, SelectInput, SelectList, SelectItem } from "@rent_avail/select"
```

```jsx live
function SelectExample() {
  const options = [
    { label: "Alabama", value: "AL" },
    { label: "Alaska", value: "AK" },
    { label: "Arizona", value: "AZ" },
    { label: "Arkansas", value: "AR" },
    { label: "California", value: "CA" },
    { label: "Colorado", value: "CO" },
    { label: "Connecticut", value: "CT" },
  ]
  // This is funny cause it's truly state state.
  const [state, setState] = useState(options[0].value)
  return (
    <Fragment>
      <p>Selected State: {state}</p>
      <Select
        id="select-id"
        onSelect={(value) => setState(value)}
        defaultValue={options[0].value}
      >
        <SelectInput label="Choose a state" />
        <SelectList>
          {options.map(({ label, value }) => (
            <SelectItem key={value} value={value} label={label}>
              {label}
            </SelectItem>
          ))}
        </SelectList>
      </Select>
    </Fragment>
  )
}
```

### Description

Select inputs display the currently selected option above a menu that lets the user choose from a list of pre-defined values.

#### Style Props

`space`

#### Select Props

| Name         | Type       | Description                                                             |
| ------------ | ---------- | ----------------------------------------------------------------------- |
| id           | `string`   | Unique identifier for the select group.                                 |
| defaultValue | `string`   | Component value will be this on mount if present.                       |
| onSelect     | `function` | Callback function that fires when a user selects an item from the list. |
